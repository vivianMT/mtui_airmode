MTUI_AIRMODE 
PROJECT OVERVIEW
MTUI_AIRMODE is a web‑based airline management system developed using PHP, MySQL, HTML, CSS, and JavaScript. It provides two main roles: Admin and Passenger, enabling efficient management of flights, bookings, payments, and tickets.

FEATURES
- Flight management  
- Passenger bookings  
- Payment processing and updates  
- Ticket generation (PDF) and sending  
- Secure authentication system  

ADMIN
- Manage flights (add, edit, delete)  
- View and manage passenger bookings  
- Handle passenger messages  
- Update booking payments  

PASSENGER
- Browse available flights  
- Make and process bookings  
- Handle and update payments  
- Select seats  
- View and download tickets (PDF)  
- Send tickets via system/email  

TECHNOLOGIES USED
- PHP – Server‑side scripting  
- MySQL – Database management  
- HTML – Page structure  
- CSS – Styling and layout  
- JavaScript – Client‑side interactivity  

DATABASE SETUP
1. Open phpMyAdmin.  
2. Create a new database (e.g., mtui_airmode).  
3. Import mtuiairmodedb.sql.  

INSTALLATION
1. Clone the repository:
   `bash
   git clone https://github.com/vivianMT/MTUI_AIRMODE.git
   `
2. Move the project folder to your server directory (htdocs for XAMPP).  
3. Configure database credentials in config/db.php.  
4. Start Apache and MySQL servers.  
5. Access the system at:
   http://localhost/mtui_airmode
   
SYSTEM FLOW
1. Passenger registers/logs in → browses flights → books a ticket → makes payment → selects seat → downloads/sends ticket.  
2. Admin logs in → manages flights → reviews bookings → updates payments → responds to passenger messages.
